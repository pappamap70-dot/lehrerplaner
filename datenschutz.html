Welche Daten werden gespeichert?

✓ Nur lokal auf deinem Gerät

Alle Daten werden ausschließlich lokal im Browser gespeichert (IndexedDB). Es gibt keinen Server, keine Cloud und keine Weitergabe an Dritte.

Folgende Daten können in der App erfasst werden:

  • Klassenbezeichnungen
  • Schülernamen
  • Noten und Leistungsbewertungen
  • Anwesenheitsdaten
  • Bemerkungen und Notizen
  • Kalendereinträge und Termine

Backup-Dateien (.lpe) werden verschlüsselt exportiert und liegen ausschließlich dort, wo du sie selbst speicherst. Ohne dein Passwort sind sie für niemanden lesbar.

Google Kalender Integration

Optional — nur auf ausdrücklichen Wunsch

Die Google-Kalender-Synchronisation ist deaktiviert, bis du sie selbst einrichtest. Sie ist vollständig optional.

Wenn du die Synchronisation aktivierst, gilt:

  • Synchronisiert werden nur: Termine, Schultermine, Konferenzen und Klassenarbeiten
  • Niemals synchronisiert: Schülernamen, Noten, Anwesenheitsdaten, Bemerkungen
  • Google erhält keinerlei Schüler- oder Leistungsdaten
  • Die Verbindung kann jederzeit in den Einstellungen getrennt werden

Für die Verarbeitung deiner Daten durch Google gelten zusätzlich die Datenschutzbestimmungen von Google ↗ .

Keine Weitergabe an Dritte

Es findet keinerlei Weitergabe von Daten an Dritte statt. Die App enthält:

  • Kein Tracking
  • Keine Werbung
  • Keine Analyse-Tools (kein Google Analytics, kein Matomo o. Ä.)
  • Keinen externen Server

Deine Rechte

Du hast jederzeit die volle Kontrolle über deine Daten:

  • Alle Daten können in den Einstellungen → Alle Daten löschen vollständig entfernt werden
  • Backups können exportiert und danach aus dem Browser gelöscht werden
  • Die Google-Kalender-Verbindung ist jederzeit trennbar — deine Daten bleiben dabei in der App erhalten

Hosting über GitHub Pages

Die App-Dateien (HTML, CSS, JavaScript — kein Inhalt, keine Nutzerdaten) werden über GitHub Pages ↗ bereitgestellt.

GitHub kann dabei technische Zugriffsdaten (IP-Adressen, Zeitstempel) protokollieren. Weitere Informationen findest du in der Datenschutzerklärung von GitHub ↗ .

Deine eigentlichen Daten (Schüler, Noten, Termine usw.) verlassen deinen Browser zu keiner Zeit und werden niemals an GitHub übertragen.
